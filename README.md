## ❓ FAQ

**Q: Is there a risk of getting my system corrupted?**  
A: No. The tool creates a system restore point automatically before any cleaning operation. You can always roll back changes using the built-in backup manager or Windows System Restore.

**Q: The scan takes a long time. Is this normal?**  
A: The first scan may take up to 10 minutes depending on your disk size and the number of files. Subsequent scans are faster because the tool caches previous results.

**Q: How often should I run PC Fresh?**  
A: For best performance, we recommend running it once a week for quick cleanup and once a month for a full system scan. Avoid running it more than once per day.

**Q: Does it clean browser passwords or saved logins?**  
A: By default, it only clears cache, cookies, and history. If you want to clear saved passwords, you need to manually enable that option in the browser cleaner settings.

**Q: Can I use it alongside another antivirus?**  
A: Yes, PC Fresh is a system utility, not an antivirus. It works alongside any antivirus software without conflicts.
